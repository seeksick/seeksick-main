## 🎯 사용법

## 📊 확인 사항

### ✅ 정상 작동 확인

1. **연결 상태**: 헤더에 "실시간 연결됨" (초록색 점)
2. **감정 바**: 애니메이션되며 업데이트
3. **챗봇**: 메시지 전송 시 AI 응답 수신

### ⚠️ 문제 발생 시

#### 모델 로드 실패?
```
⚠️ 모델 로드 실패 (데모 모드로 실행)
```
→ **정상입니다!** 데모 모드로 작동하며, 랜덤 감정 데이터를 사용합니다.

#### ChatGPT 응답 없음?
→ API 키 확인: `app.py` 68번째 줄

#### 포트 충돌?
```bash
# 다른 포트로 실행
export FLASK_PORT=5001
python3 app.py
```
